## :wrench: Installation

### :computer: Web

```
  // 1 - Git Clone
  git clone https://github.com/TiagoBehenck/github-explorer

  // 2 - Installing the dependencies
  cd github-explorer && yarn install

  // 3 - Run the application
  yarn start

```
